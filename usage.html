How to Use FILAN

  1. Download FILAN from GitHub Releases.
  2. Run the app on both sender and receiver machines.
  3. Enter the ID displayed on the other device to connect.
  4. Select files to send and press "Transfer".

Quick Command

python filan.py

After launch, you'll see your device's FILAN ID. Share it with the other party, and start transferring files!
