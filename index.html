FILAN

FILAN is a lightweight, cross-platform LAN file sharing tool. Share files quickly over your local network without IP hassle — just use an ID.

Features

  • Fast and simple LAN file transfer
  • ID-based peer discovery
  • Cross-platform (Windows, Mac, Linux)
  • No external configuration required
